## Installation <a name="install"></a>
You can see the commands we use frequently in the makefile in details. <a href="https://github.com/Umut-Deniz1/crypto_api/blob/master/Makefile">(Makefile)</a>

    $ git clone project
    $ cd crypto_api
    $ make build
    $ make up
